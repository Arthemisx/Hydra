# Hydra

Projeto Integrador feito em parceria com a Faculdade São Camilo.
A partir de reuniões feitas com o parceiro tivemos a oportunidade de entender o problema por eles enfrentado e assim, através de sprints semanais com os membros do grupo e a utilização de cartões de user story para uma melhor compreensao sobre as questões abordadas.
Desse modo, desenvolvemos um aplicativo multiplataforma que regsitra informações sobre o dia a dia de atletas e transforma em formulario para que treinadores e nutricionistas possam acompanhar de maneira facil a rotina de seus alunos e fazer ajustes sempre que necessar.
